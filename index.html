About

An Introduction!

Flow Android App is a power application to send and receive data inside the mines through vehicles.

Flow is an android application that wirelessly beams information on a vehicleâ€™s GPS location, mileage, fuel usage, operating conditions, and driver behavior and more to the cloud without the need for third-party plug-in devices. The CAN data is collected from vehicle and send to the IFM device using bluetooth which later upload the data packages to the cloud using wifi.

The data package include the information of bluetoot devices, access points, accelerometer, CAN data and beacons. The data can be analyzed in such a way to track the direction of the vehicle by installing the hardware products owned by Millescond Oy including IFM devices, antennas horizontally polarized etc.

Installation Steps

Steps

Step 01: Installing the application

  • Press "INSTAL" on question 'Do you want to install this application ?'.
  • Press "Open".

    • Step 02: Configuring the application

  • Press "ALLOW" to give access to photos, media and files.
  • Press "ALLOW" to give access to location of device.

    • Step 03: Connecting to application

  • Enter you email address on 'Welcome Screen' to get the PIN code.
  • Press 'Already have code' to enter the PIN code received in the email.
  • Press "Send" after entering the PIN.

    • Step 04: Understanding the application

  • The 'green' color of "Service" and "CAN device" show the connectivity. If it is 'red' then there is no connected device.
  • Press 'Dashboard' to see the content.
  • Press 'User form' to fill-in and send the user form to backend.
  • Press 'Setting -> About' to see the application version, configuration file, ID, and account type.
